### Большой проект по интерпреации АВ тестирования: AB_testing_results.ipynb

#### Контекст: 

В дейтинговом приложении доход во многом формируется за счёт премиум-подписки. Команда продукта протестировала новую ценовую политику: для части новых пользователей из ряда стран изменилась стоимость премиума при оплате через две новые платёжные системы (стоимость пробного периода не менялась).

Цель бизнеса:
> Понять, был ли эксперимент успешен.

Здесь: 

- Проведен подробный EDA;
- Выбраны метрики для отслеживания результатов;
- Произведен расчет метрик и проверка их статистической значимости;
- Написан подробный отчет о проделанной работе.

### Основные библиотеки: 

- matplotlib
- pingouin
- pandas
- numpy
- statsmodels

### Большой проект по продуктовой аналитике: Brazilian_marketplace_product_analysis.ipynb

#### Контекст: 

Есть стартап, который создает новый маркетплейс. Он недавно появился на рынке и занимается продажей новых товаров из Бразилии, которые только начинают поступать в продажу.
Продакт-менеджер  переживает за свой продукт, так как выручка маркетплейса стоит на месте уже несколько месяцев. Он предложил мне полную свободу действий. Главное — чтобы метрики росли, а мы не причиняли неудобства клиентам.  Продажи идут плохо, надо выяснить причину.

### Использовалось:

- Когортный анализ
- Выбор продуктовых метрик
- pandas
- matplotlib
- PMF
- ICE framework
- Выбор гипотезы
